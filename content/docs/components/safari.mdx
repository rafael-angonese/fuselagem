## Props

| Attribute | Type     | Description                                                 | Default     |
| --------- | -------- | ----------------------------------------------------------- | ----------- |
| url       | `string` | The URL to display in the Safari address bar                | `undefined` |
| src       | `string` | The source URL of the image to display in the Safari window | `undefined` |
| width     | `number` | The width of the Safari window                              | `1203`      |
| height    | `number` | The height of the Safari window                             | `753`       |
