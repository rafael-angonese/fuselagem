# Installation

Requirements:

- [Node.js 20.15.1](https://nodejs.org/) or later installed on your machine.

---

## Installation

<Steps>

#### Clone the Repository

```codeBlock bash
git clone git@github.com:rafael-angonese/docs.git

```

#### Navigate to the Project Directory

```codeBlock bash
cd your-project

```

#### Install Dependencies

<PackageManagers
  commands={{
    npm: "npm install",
    yarn: "yarn",
    pnpm: "pnpm install",
    bun: "bun install"
  }}
/>

#### Start the Application

<PackageManagers
  commands={{
    npm: "npm run dev",
    yarn: "yarn dev",
    pnpm: "pnpm dev",
    bun: "bun dev"
  }}
/>

#### Open in Browser

Open your web browser and navigate to [http://localhost:3000](http://localhost:3000) to view the application.

</Steps>
